LaundroMate

Categories

Research | App Design | Presentation Design

See the creation of a conceptual phone application that helps solve issues in current shared laundry facilities.

Team Members

Sophie & Jack

Date Created

April 2019

Project Goal

While in college, we were tasked with studying users of public/shared laundry facilities and design an application that works with a conceptual product using IoT technology to solve some of their common issues.

Initial Research

To find solutions for these issues, we needed to learn more about them. During our study, our target audience was narrowed to users of shared laundry facilities within apartment buildings and school residences, so we can create the best product for our users.

View research report

Proposal

Proposal

We propose to develop LaundroMate, a self-serve laundry machine app that allows self serve laundry users to connect to a smart combination washer-dryer via Bluetooth and gather information about a facilityâ€™s machines.

The app will allow:

  • Users to see if there are machines available
  • Users to book a machine before arriving to laundry facility
  • Users to lock machines to prevent theft or damage
  • Users to pay for the service electronically

App Design

App Design

Flowchart

Before designing the app, we had to consider
the main tasks that our users would have to complete when using the application.

The main tasks are:

  • Sync apartment to the app
  • Reserve a laundry machine
  • Unlock a machine for use
  • Configure the laundry machine
  • Add another laundry load
  • Approve late pick-up fee
  • Unlock a machine for pick-up
  • Request notifications for the next
    available machine
View complete flowchart

Sketches

Each screen from start to finish of every task had to be designed. Sketches were created for each screen to ensure every element that is needed is considered in the layout.

Usability Testing: Focus Group

After planning the screens for each task, we met with a group to test the intuitiveness of the steps. Any issues that arose were documented and changes were considered for a smoother process.
