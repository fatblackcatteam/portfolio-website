Our Process

1

Requirement Gathering

Once an estimate is accepted, we will request any brand guidelines you may have, then we will gather all information and content needed to begin the project.

Text and content for the project should be finalized before the project is started.

Changes in content can drastically change the layout of a project.

2

Style Exploration

During Style Exploration, we will present you with some previews of a portion of the project with different style options to be narrowed down to a final style.

3

Initial Designs

Once a style established, we will present you with the first draft of the project. We will then discuss any edits that may be needed before the final version.

4

Revisions

Similarly to the last step, during this step, we will discuss if any further changes are needed. This step may need to be repeated a few times.

5

Final Deliverables

All elements of the project are saved and prepared for all applications. This is the final step of the project. Changes to the project cannot be made after this step without opening a new project.
